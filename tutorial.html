Sigefica - Tutorial de uso

Página Inicial

Campo de Dados Pessoais

Instruções

  • Nome → Digite seu nome, deixando de fora apenas o último sobrenome, exemplo: João Pedro de Oliveira → João Pedro de
  • Sobrenome → Aqui sim você digita seu sobrenome, exemplo: João Pedro de Oliveira → Oliveira
  • Email → Informe um email, esse email será usado para o envio da ficha catalográfica revisada. Certifiquisse de digita-lo corretamente, pois, você receberá a ficha por emaail.

Campo de Dados do Trabalho

Instruções

  • Título → Digite o título completo do trabalho
  • Tipo de trabalho → Selecione a opção que condiz com o tipo do seu trabalho
  • Curso → Selecione seu curso
  • Instituição → Informe o nome da sua instituição
  • Cidade → Digite a cidade onde fica a instituição
  • Ano de publicação → Selecione o ano em que seu trabalho será publicado
  • Número de páginas → Digite a quantidade de páginas que seu trabalho possui

Campo de Palavras-Chave

Instruções

  • Campos → Cada campo equivale a SOMENTE UMA palavra-chave
  • Quantidade → A quantidade mínima de palavras-chave são 3, e a máxima são 5 → Caso seu trabalho possua menos de 5, deixar os demais campos em BRANCO

Campo de Dados do Orientador

Instruções

  • Nome do Orientador → Digite o nome completo do orientador do seu projeto
  • Titulação → Selecione a opção que condiz com a titulação do seu orientador

Gerar Ficha

Gerar ficha → Pronto! Clique neste botão para verificar se nenhum campo está em branco, caso tenha preenchido tudo, o sistema gerará a ficha e enviará para a bibliotecária! Fique atento ao seu email para receber o retorno da ficha!
