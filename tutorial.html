Sigefica - Tutoria de uso

Página Inicial

Área → Informações Básicas

Instruções

  • Nome → Digite seu nome, deixando de fora apenas o último sobrenome, exemplo: João Pedro de Oliveia → João Pedro de
  • Sobrenome → Aqui sim você digita seu sobrenome, exempl: João Pedro de Oliveira → Oliveira
  • Título → Digite o título completo do trabalho
  • Tipo de trabalho → Selecione a opção que mais condiz com o tipo do seu trabalho
  • Curso → Selecione seu curso
  • Email → Informe um email, esse email será usado para o envio da ficha catalográfica revisada

Área → Dados do Orientador

Instruções

  • Nome do Orientador → Digite o nome do orientador do seu projeto
  • Titulação → Selecione a opção que condiz com a titulação do seu orientador

Área → Palavras Chave

Instruções

  • Campos → Cada campo equivale a SOMENTE UMA palavra chave
  • Quantidade → A quantidade mínima de palavras chave é 1, e a máxima é 5 → Caso seu trabalho possua menos de 5, deixar os demais campos em BRANCO

Área → Palavras Chave

Instruções

  • Cidade → Digite a cidade onde fica a instituição
  • Ano de publicação → Selecione o ano em que seu trabalho será publicado
  • Número de páginas → Digite a quantidade de páginas que seu trabalho possui
  • Gerar ficha → Pronto! Clique neste botão para verificar se nenhum campo está em branco, caso tenha preenchido tudo, o sistema gerará a ficha e enviará para a bibliotecária! Fique atento ao seu email para receber o retorno da ficha!
